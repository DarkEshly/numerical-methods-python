# Численные методы на Python
Этот репозиторий содержит набор численных методов, реализованных на Python в рамках университетского проекта. Проекты демонстрируют фундаментальные алгоритмы для решения систем линейных и дифференциальных уравнений. 
## gauss.py
Принимает на ввод матрицу в системе линейных алгебраических уравнений Ax = f и выводит следующие значения: решение системы методом Гаусса, решение системы методом Гаусса с выбором главного элемента, рпределитель матрицы, обратнуюматрицу, число обусловленности матрицы.
