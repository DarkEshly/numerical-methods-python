# Численные методы на Python
Этот репозиторий содержит набор численных методов, реализованных на Python в рамках университетского проекта. Проекты демонстрируют фундаментальные алгоритмы для решения систем линейных и дифференциальных уравнений. 
## gauss.py
Принимает на ввод матрицу и вектор правой части  в системе линейных алгебраических уравнений *Ax = f* и выводит следующие значения: решение системы методом Гаусса, решение системы методом Гаусса с выбором главного элемента, рпределитель матрицы, обратнуюматрицу, число обусловленности матрицы.
## relax.py
Принимает на ввод матрицу и вектор правой части в системе линейных алгебраических уравнений *Ax = f*, а также итерационный параметр и выводит следующие значения: решение системы методом верхней релаксации, количество итераций цикла,
затраченных на нахождение решения заданной точности.
## runge-kurt.py
Демонстрирует работу методов для решения обыкновенных дифференциальных уравнений первого порядка. Спрашивает у пользователя номер теста, который нужно продемонстрировать, параметр n - число точек приближённого вычисления и
метод решения - метод Рунге-Кутта второго или четвёртого порядка. Программа затем выводит таблицу сравнения приближённого и
точного решения и показывает соответствующие графики.
## edge_problem.py
Демонстрирует работу методов для решения дифференциальных уравнений второго порядка. Спрашивает у пользователя номер теста, который нужно продемонстрировать, параметр n - число точек приближённого вычисления. Программа затем выводит таблицу сравнения приближённого и точного решения и показывает соответствующие графики.
